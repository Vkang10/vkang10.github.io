L’OREAL PRINT AD
DESIGNS OPTIMIZATION

(SAS, Excel)

summary

To attract consumers’ attention in a complicated media bombard environment, this project assessed the effectiveness and optimized the design of L’Oreal ads in order to increase brand recognition and create a durable memory of the brand.

Through analysis of eye-tracking data including 88 participants and 35 different advertisements, such as fixation count and surface size, we developed Poisson regression and Binary logit model to find the correlation between variables, such as page number, page position, brand size, pic size, etc. By evaluating BIC, we choose the optimal binary logit model(BIC:4070.328) and put the parameters in excel to find the most suitable pictorial and brand size of ads, improving the predicted recall accuracy value to 0.398.

Model

/* Poisson Regression Models */

/* For BRAND_FIX: fixation counts of the brand element */ 
proc genmod data=temp;
     model brand_fix=brand_size page_pos page_num loreal brandlength/
           dist=poisson link=log;
     title 'Poission Regression for brand fixation count';

/* For PIC_FIX: fixation counts of the pictorial element */ 
proc genmod data=temp;
     model pic_fix=pic_size page_pos page_num loreal/
           dist=poisson link=log;
     title 'Poission Regression for pictorial fixation count';

/* Binary Logit Model */
proc genmod data=temp descending;
     model recall_accu=brand_fix pic_fix page_pos page_num loreal RECALL_TIME/
	       dist=binomial link=logit;
     title 'Binary Logit Model for Recall Accuracy = 1';
run;
